## 陣列及記憶體

在C語言中，陣列的開頭位置即為陣列第一個元素的位置

```
#include <stdio.h>
int main(){
    int arr[5] = {1,2,3,4,5};
    if(arr == &(arr[0])){
        printf("true");
    }
    return 0;
}
```

因此，我們可以利用下列方法來求陣列儲存的值

```
int arr[5] = {1,2,3,4,5};
printf("%d",*(arr+3)); //取arr[3]
```

## 多維陣列

在有時，我們想要表現多維度的資料，此時我們就會使用多維陣列，宣告方法如下

```
int arr[3][5];
```

操作方法類似於一維陣列，以下面程式碼為例

```
#include <stdio.h>

int main(){
    int arr[3][5]; 
    for(int i = 0;i<3;i++){
        for(int j = 0;j<5;j++){
            scanf("%d",&arr[i][j]);
        }
    }
    for(int i = 0;i<3;i++){
        for(int j = 0;j<5;j++){
            printf("%d ",arr[i][j]);
        }
        printf("\n");
    }
    return 0;
}
```
